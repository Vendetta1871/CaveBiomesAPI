# CaveBiomesAPI

Мод для Minecraft 1.12.2, добавляющий изменяемую высоту мира и систему 3D-биомов.

## Автор
CelestialD

## Пакет
net.celestiald.cavebiomes

## Глобальный план
- Этап 1: Изменяемая высота мира (конфиг + API для других модов)
- Этап 2: 3D-биомы (подземные биомы с вертикальной координатой)

## Правила
- В сообщениях коммитов НИКОГДА не упоминать Claude, AI, ИИ или что-либо подобное.
- Пакет всегда: net.celestiald.cavebiomes
- Имя автора во всех файлах проекта: CelestialD

## Стек
- Minecraft 1.12.2 / Forge 14.23.5.2864
- SpongePowered Mixin 0.7.11-SNAPSHOT
- Java 8
